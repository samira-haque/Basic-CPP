## 📖 About

**Basic C++** is a comprehensive learning repository designed for beginners who want to build a strong foundation in C++ programming. It covers fundamental programming concepts, control structures, arrays, functions, object-oriented programming, and problem-solving techniques through well-organized examples and practice programs. Whether you're a student, a programming enthusiast, or preparing for coding interviews, this repository provides a structured path to mastering C++.

## 🚀 Getting Started

### Prerequisites

- C++ Compiler (GCC/G++)
- Code::Blocks, VS Code, Dev-C++, or any C++ IDE

### Clone the Repository

```bash
git clone https://github.com/samira-haque/Basic-CPP.git
cd Basic-CPP
```

### Compile and Run

```bash
g++ filename.cpp -o output
./output
```

#### Windows

```bash
g++ filename.cpp -o output.exe
output.exe
```
